PIG JSON-LD Schemata

Product Information Graph - JSON Schema Validation (Draft-07)

Available Schemata

PIG Metamodel Class Items

The classes are derived from the ontology.

  • Class Property.json
    Validates Property class definitions with datatypes, constraints, and enumeration values
  • Class Link.json
    Validates Link class definitions with eligible endpoints
  • Class Entity.json
    Validates Entity class definitions with eligible properties and target links
  • Class Relationship.json
    Validates Relationship class definitions with source and target link specifications

PIG Metamodel Instance Items

The instances of the classes above are the project data or 'payload'.

  • Instance anEntity.json
    Validates entity instances with configurable properties and target links
  • Instance aRelationship.json
    Validates relationship instances with source/target links and properties

PIG Package Item

A package holds a subset of the graph for a specific communication purpose such as customer-supplier agreement.

  • Package aPackage.json
    Validates complete JSON-LD documents with @context and @graph

Key Features

  • JSON Schema Draft-07 compliant
  • Strict validation with additionalProperties: false
  • Pattern-based property validation for configurable items
  • Multi-language text support with IETF language tags
  • Conditional schema application with if-then logic
  • Modular schema design with reusable definitions
  • Complete package validation with @graph
